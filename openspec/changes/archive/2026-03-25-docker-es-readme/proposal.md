## Why

需要为项目提供清晰的Docker化Elasticsearch部署指南，方便开发者和用户快速搭建搜索服务环境。当前项目缺乏容器化部署文档，导致环境配置效率低下。

## What Changes

- 创建README.txt文件，包含从Docker拉取和运行Elasticsearch的完整指南
- 提供基础配置和常用命令示例
- 包含故障排查和验证服务运行的步骤

## Capabilities

### New Capabilities
- docker-es-guide: 提供完整的Docker Elasticsearch部署和使用指南

### Modified Capabilities
- 无现有功能修改

## Impact

- 新增文档文件README.txt
- 不影响现有代码和API
- 无依赖项变更
